# Capabilities — things the agent can do that it will forget

This file is loaded into the L1 hot memory block every session. Add capabilities here as they are built. Remove when decommissioned. One line per capability.

## Example entries (replace with your own):
- Voice output via smart speaker (connected, text-to-speech)
- Media control (playlists, tracks, volume, pause)
- Home automation control via HA integration
- Wake on LAN for remote machines
- Git-based issue tracking and review pipeline
- Sensor data from apartment (door, presence, temperature, power)
- Self-hosted services on local network
